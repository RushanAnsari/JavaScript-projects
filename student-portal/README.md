# 🎓 Student Portal (Login & Registration UI)

A modern and responsive **Student Registration & Login System UI** built using **HTML, CSS, Bootstrap, and JavaScript**.

---

## 🚀 Features

* ✨ Modern UI Design (Bootstrap आधारित)
* 📱 Fully Responsive Layout
* 🎯 Two-column Registration Form
* 🔐 Show/Hide Password Toggle
* ⚡ Real-time Form Validation (Regex)
* 🔑 Password Strength Indicator
* 🎨 Smooth Animations & Stylish Inputs
* 🔗 Login & Register Page Navigation

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript (Vanilla JS)

## 🧪 How to Run

1. Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/javascript-projects.git
```

2. Navigate to project folder:

```
cd javascript-projects/student-portal
```

3. Open in browser:

```
register.html
```
